About This Demo

This page demonstrates the NLWeb Chat Widget, which provides:

  • Full conversation management with sidebar
  • Remembered items functionality
  • Context-aware responses using previous queries
  • Streaming responses with real-time updates
  • Persistent conversation history (stored in localStorage)

Widget Configuration

The widget is initialized with the following configuration:

const chatWidget = new NLWebChatWidget({
  searchInputId: 'demo-search-input',
  searchButtonId: 'demo-search-button',
  containerId: 'nlweb-chat-container',
  site: 'seriouseats',
  position: 'relative'  // or 'fixed' for floating widget
});
