NLWeb Dropdown Chat Integration Example

This page demonstrates how to integrate the NLWeb Dropdown Chat component into your website

Recipe Search Example

Try searching for recipes using natural language queries:

Example queries:

  • Chocolate cake recipes
  • Spicy vegetarian dishes
  • Quick pasta recipes under 30 minutes

Integration Code:

const recipeSearch = new NLWebDropdownChat({
    containerId: 'recipe-search-container',
    site: 'seriouseats',
    placeholder: 'Search for recipes...'
});

Product Search Example

Another instance with different configuration:

Features

  • Multiple instances on same page
  • Site-specific search filtering
  • Conversation history per site
  • Follow-up questions support
  • Clean, minimal integration

Full Integration Example:

<!-- Include CSS -->
<link rel="stylesheet" href="nlweb-dropdown-chat.css">

<!-- Add container -->
<div id="my-search-container"></div>

<!-- Include JavaScript -->
<script type="module">
  import { NLWebDropdownChat } from './nlweb-dropdown-chat.js';
  
  const chat = new NLWebDropdownChat({
    containerId: 'my-search-container',
    site: 'all',
    placeholder: 'Ask anything...',
    endpoint: 'http://localhost:8000'
  });
</script>
