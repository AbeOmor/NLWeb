About the Web Component

The <nlweb-chat> web component is a fully encapsulated chat interface that can be embedded into any website with a single HTML tag.

Basic Usage

<!-- Include the web component script -->
<script src="/static/nlweb-chat-element.js"></script>

<!-- Add the component to your page -->
<nlweb-chat 
  search-input-id="my-search-input"
  search-button-id="my-search-button"
  site="seriouseats"
  position="relative">
</nlweb-chat>

Features

  • Shadow DOM encapsulation - no style conflicts
  • Full conversation management with sidebar
  • Remembered items functionality
  • Context-aware responses
  • Persistent storage (localStorage)
  • Streaming responses
  • Custom events for integration
  • Responsive design

Attributes

  • search-input-id: ID of the search input element
  • search-button-id: ID of the search button element
  • site: Site to search (e.g., 'seriouseats', 'all')
  • position: 'relative' (dropdown) or 'fixed' (floating)

JavaScript API

show()

Opens the chat interface

chatElement.show();

close()

Closes the chat interface

chatElement.close();

toggle()

Toggles the chat interface open/closed

chatElement.toggle();

search(query)

Programmatically perform a search

chatElement.search('chocolate cake recipes');

setQuery(query)

Sets the search input value without searching

chatElement.setQuery('pasta recipes');

Custom Events

// Listen for chat opened
chatElement.addEventListener('nlweb-chat-opened', (e) => {
  console.log('Chat opened');
});

// Listen for messages sent
chatElement.addEventListener('nlweb-message-sent', (e) => {
  console.log('Message sent:', e.detail.query);
});

// Listen for messages received
chatElement.addEventListener('nlweb-message-received', (e) => {
  console.log('Response received:', e.detail);
});
